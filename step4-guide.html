B
Create the clips table in Supabase
1
Open SQL Editor in Supabase → New query
Paste and run the SQL below. This creates the table where each clip gets stored.
Creates the clips tableSQL
CREATE TABLE IF NOT EXISTS clips (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  video_id uuid REFERENCES videos(id) ON DELETE CASCADE,
  title text,
  summary text,
  transcript_text text,
  start_ms integer,
  end_ms integer,
  duration_secs integer,
  source text, -- 'chapter', 'highlight', 'sentiment'
  score integer DEFAULT 0,
  created_at timestamptz DEFAULT now()
);

-- Allow users to read clips from their own videos
ALTER TABLE clips ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Users read own clips"
  ON clips FOR SELECT
  USING (
    video_id IN (SELECT id FROM videos WHERE user_id = auth.uid())
  );

-- Also add transcript_id column to videos table
ALTER TABLE videos ADD COLUMN IF NOT EXISTS transcript_id text;
2
Click Run → success message appears
Go to Table Editor — you should now see both videos and clips in the list.

Add secret keys to Vercel
1
Go to your Vercel project settings
On vercel.com ↗ → click your ClipVault project → Settings tab → Environment Variables.
2
Add these 3 variables
Click Add New for each one:
Variable nameValue (where to find it)
ASSEMBLYAI_API_KEY AssemblyAI dashboard → your API key
SUPABASE_URL Supabase → Settings → API → Project URL
SUPABASE_SERVICE_KEY Supabase → Settings → API → service_role key (secret!)
3
Set environment to "All" and save each one
Make sure the dropdown says Production, Preview, Development for each variable so it works everywhere.
The service_role key has full database access — only ever put it in Vercel environment variables, never in your HTML files.

F
Troubleshooting
?
Card stays "Processing…" forever
Check Vercel → Functions tab → see if process-video ran and what error it logged. Most common cause: wrong SUPABASE_SERVICE_KEY or ASSEMBLYAI_API_KEY in env vars.
?
AssemblyAI returns 0 clips
Very short videos (<15 seconds) may not generate chapters. Try a 60-second video with speech. AssemblyAI needs audio — music-only videos won't clip.
?
Function timeout error
Long videos (>10 min) may exceed Vercel's 60-second function limit on the free plan. Solution: upgrade to Vercel Hobby ($0, 5-minute limit) or Pro ($20/mo, unlimited).
